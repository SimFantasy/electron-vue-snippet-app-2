# SimSnippet

## 产品功能

### 搜索框

- 输入关键字查询出符合关键字的代码片段结果列表
  - 通过键盘方向键（上下键）在列表中移动选择需要的代码片段，回车将代码片段内容复制到剪贴板，并隐藏搜索框
  - 通过鼠标点击结果列表中的代码片段也可以将点击的代码片段内容复制到剪贴板，并隐藏搜索框
- 点击搜索框上的管理按钮，显示代码片段管理窗口

### 代码片段管理

#### 头部功能区： 

- 标题：显示应用标题和图标
- 操作区： 
  - 设置按钮： 点击在管理窗口弹出设置弹窗
  - 窗口最小化按钮： 点击最小化代码片段管理窗口
  - 窗口最大/恢复按钮： 点击切换最大化和恢复窗口大小
  - 窗口关闭： 点击隐藏代码管理窗口

#### 主功能区：

##### 左侧分类管理：

- 快捷导航：
  - 全部代码片段：显示所有代码片段信息
  - 未分类： 显示所有未分类的代码片段信息
  - 收藏夹： 显示所有收藏的代码片段
  - 回收站： 
    - 显示所有回收站中的代码片段
    - 鼠标悬停到导航上，显示更多按钮，点击显示： 清空回收站 菜单，点击清空回收站菜单清空回收站中所有软删除的代码片段
    - 在导航上点击右键，显示 清空回收站 菜单，点击 清空回收站 菜单，清空回收站中所有软删除的代码片段
- 分类导航：
  - 显示所有代码片段分类
  - 分类按创建时间倒序排序
  - 点击分类，显示该分类下的所有代码片段列表
  - 鼠标悬停到分类导航上，显示更多按钮，点击显示：
    - 重命名： 修改分类导航的名称
    - 删除分类： 删除该分类
      - 如果分类下没有代码片段，直接删除，删除后提示删除成功
      - 如果分类下有代码片段，提示： 删除后，该分类下的所有代码片段将会转为未分类，确定则执行，成功后提示删除成功
    - 添加代码片段： 在该分类下添加代码片段
  - 拖拽分类导航，可以对分类导航进行排序
- 标签导航：
  - 显示所有标签
  - 点击标签，显示该具有改标签的所有代码片段

##### 中间代码片段列表：

- 头部
  - 搜索框： 根据关键字，搜索分类下的所有代码片段
  - 添加代码片段按钮： 点击添加代码片段
- 代码片段列表
  - 显示代码片段名称，分类、 标签、 更新日期
  - 代码片段按 更新时间倒序排序
  - 在代码片段item上点击右键，显示右键菜单：
    - 修改分类： 通过右键二级菜单显示所有分类，点击对应的分类，将当前的代码片段分类修改为选择的分类
    - 删除代码片段： 
      - 如果当是普通分类，点击删除代码片段，将代码片段移动到回收站。 
      - 如果当前分类已经是回收站了，则弹出删除确认提示框，如果确认删除，则执行硬删除，删除成功提示删除成功
    - 添加/取消收藏：
      - 如果当前已经收藏了，则显示取消收藏代码片段，点击取消该代码片段的收藏
      - 如果当前未收藏，则显示收藏代码片段，点击收藏该代码片段
  - 代码片段列表每页显示50条，如果往下滚动则自动加载后续分页内容

##### 右侧代码片段详情：

- 代码片段详情是代码片段添加、更新的表单
- 代码片段名称：
  - 文本框，显示代码片段完整名称
  - 当添加代码片段时，显示未命名代码片段
  - 当修改代码片段名称时，中间代码片段列表中的该代码片段的名称也需要实时修改
- 分类：
  - 下拉列表，显示所有代码片段分类
  - 点击下拉列表中的分类，则对代码片段分类进行修改
  - 当添加代码片段时，分类为未分类
  - 当修改代码分类后，中间代码片段列表中的该代码片段的分类也需要实时更新
- 标签：
  - 标签输入框， 显示多个标签，标签统一为小写，在输入框中输入后回车，则将输入内容转为标签
  - 添加代码片段时，标签为空，显示输入提示
  - 当更新代码片段时，中间代码片段列表中的该代码片段的标签也需要实时更新
- 代码内容
  - codemerror代码编辑器
  - 当添加代码片段时，显示为空
- 代码语言：
  - 下拉列表，默认显示设置中配置的默认代码语言
  - 选择下拉列表中的代码语言，可以修改代码内容中codemerror代码编辑器中的代码渲染
- 代码统计：
  - 当前代码所在行数、列数
  - 代码总行数
- 代码截图（后期规划，点击后将代码片段生成截图，保存或复制到剪贴板）

### 代码设置

#### 通用设置

- 开机启动： 开关，默认关
- 应用启动时不显示搜索框： 开关， 默认开
- 搜索框失去焦点时隐藏： 开关，默认开
- 显示系统托盘图标： 开关，默认开
- 搜索框置顶： 开关，默认关
- 搜索框位置： 下拉列表， 默认居中，
  - 列表项： 居中、左上、 中上、 右上、 自定义
  - 选择自定义时，显示x和y轴文本输入框，用户自定义位置
- 语言： 下拉列表，默认 简体中文
  - 列表项： 简体中文、繁体中文、English

#### 外观

- 主题模式： 下拉列表， 默认跟随系统
  - 下拉项： 跟随系统、浅色、深色
- 主题： 图片下拉列表， 默认github-light
  - 下拉项： 'material-light'、 'github-light'、 'tomorrow'、 'solarized-light'、 'material-dark'、 'github-dark'、 'tomorrow-night'、 'solarized-dark'、 'on-dark'、 'dracula'、 'monokai'
  - 当主题模式为跟随系统时显示所有主题
  - 当主题模式为浅色时显示： 'material-light'、 'github-light'、 'tomorrow'、 'solarized-light'
  - 当主题模式为深色时显示： 'material-dark'、 'github-dark'、 'tomorrow-night'、 'solarized-dark'、 'on-dark'、 'dracula'、 'monokai'
  - 当选择主题后
    - 需要根据主题映射中的颜色信息修改整个应用的配色
    - 根据主题修改代码偏器的主题
- 搜索框： 开关，默认关闭，在关闭的情况下显示主题映射的配色
  - 背景图： 文件选择框，选择后可以使用背景图来修饰搜索框
  - 透明度： 滑块，0-1，默认1，会调整背景图上层主题映射颜色的透明度
  - 模糊度： 滑块，0-10， 默认0， 会调整背景图上层主题映射颜色的模糊度
- 管理界面背景图： 开关，默认关闭，在关闭的情况下，显示主题映射的背景色
  - 背景图： 文件选择框，选择后可以使用背景图来修饰搜索框
  - 透明度： 滑块，0-1，默认1，会调整背景图上层主题映射颜色的透明度
  - 模糊度： 滑块，0-10， 默认0， 会调整背景图上层主题映射颜色的模糊度

#### 编辑器设置

- 字体： 下拉列表，默认Fira Code，
  - 下拉列表： Fira Code', 'Source Code Pro', 'JetBrains Mono', '自定义'
  - 如果选择为自定义，则在下方显示新的下拉列表，显示本地字库，可以使用本地字库中的字体
- 字号： 下拉列表， 默认16px
  - 下拉列表： 12px、 13px、 14px、 15px、 16px、 17px、 18px、 19px、 20px、 21px、 22px、 23px、 24px
- 缩进格数：数字输入框， 默认2，最大8
- 自动换行： 开关， 默认开
- 是否高亮当前行： 开关， 默认开
- 是否显示行号： 开关， 默认开
- 代码片段默认语言： 下拉列表，默认javascript

#### 存储设置：

- 数控库存储位置： 文件选择，默认应用目录下 data.db
